Home Page

Introduction

Airline customer satisfaction is crucial for enhancing passenger experience and improving service quality. This project applies machine learning techniques to analyze customer journeys, identify satisfaction patterns, and provide insights for service improvement.

Dataset Used

We have utilized an airline customer satisfaction dataset containing:

  • Demographic Data: Gender, Age, Customer Type
  • Travel Details: Flight Distance, Type of Travel, Class
  • Service Ratings (0-5): Inflight Wi-Fi, Seat Comfort, Entertainment, Check-in Service, etc.
  • Delays: Departure and Arrival Delays

Project Workflow

  • Data Preprocessing: Cleaning, handling missing values, and standardization.
  • Dimensionality Reduction: PCA (Principal Component Analysis) to reduce redundant features.
  • Customer Segmentation: K-Means clustering to group passengers based on experience.
  • Model Evaluation: Used Accuracy Score for validation.
  • Visualization: 2D scatter plots where: Green = Satisfied Customers, Red = Dissatisfied Customers.

Observation:

Red Cluster in Plot indicates Unsatisfied Peoples.

Green Cluster in Plot indicates Satisfied Peoples.

If you observe that satisfied customers dominate higher PC1 values: "Principal Component 1 (PC1) represents a combination of features such as online booking ease, inflight services, and seat comfort etc. Higher PC1 values correspond to satisfied customers, indicating that enhancing these features could lead to higher satisfaction."

Web Application & Deployment

This project is deployed as a web application allowing users to input travel details and predict their customer segment.

  • Frontend: HTML form to collect user data.
  • Backend: Django API processing input, applying PCA & clustering.
  • Deployment: Hosted on AWS Elastic Beanstalk for scalability.

Key Benefits

  • Personalized Customer Experience: Tailors services based on customer segments.
  • Data-Driven Insights: Identifies key factors affecting passenger satisfaction.
  • Scalability & Automation: Real-time customer segmentation.

Conclusion

This project demonstrates the power of machine learning in improving airline customer experience. By analyzing customer journeys, airlines can optimize services and enhance passenger satisfaction.
